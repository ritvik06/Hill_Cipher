(a) Code-Flow

The code consists of three main python scripts - utils.py, cipher.py and cryptanalysis.py
	
	- utils.py - this contains helper functions to compute matrix modulo inverse
	- cipher.py - this contains code for part (a), i.e. encryption and decryption using hill-cipher
	- cryptanalysis.py - this contains code for part (b), i.e. cryptanalysis for hill-cipher using different key-sized and index of coincidence


(b) Instructions to run code

####### PART-A #######

Choice A :
 $ python cipher.py $location of input plain text$ $location of input key$
 
Choice B : 
 $ python cipher.py
	- enter input string given prompt
	- enter key string given prompt


####### PART-B #######

Choice A :
 $ python cryptanalysis.py $location of plain text$ $location of cipher text$

Choice B :
 $ python cipher.py
	- enter input text
	-enter the whole cipher text
